Apache Maven: Build Tool

pom - project object Model


.war - web archive
.jar - java archive
.ear - enterprise archive


sudo apt update && sudo apt upgrade -y
sudo apt install openjdk-21-jdk -y
sudo apt install maven -y


1 mvn clean:

Deletes the target folder

2 mvn validate

Pre-build sanity check.

Checks if the project is correct and all info is available.



3 mvn compile

Compiles the source code (.java to .class)

Checking for syntax errors.


4 mvn test

Runs unit tests (JUnit/TestNG).

Quality gate in Jenkins pipelines.

5 mvn package

Bundles compiled code into a JAR/WAR.

Creating the artifact for deployment.

6 mvn install

Puts the package in your Local Repository.

Sharing a library between local projects.



7 mvn deploy

Pushes the package to a Remote Repository (Nexus/Artifactory).
